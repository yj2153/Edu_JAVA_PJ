자바 프로젝트
--------------------

메신저를 활용한 온라인교육시스템
** 교육콘텐츠
* 텍스트 (쳇)
* 파일(서버 :파일제공 클라이언트 :파일다운)

** 서비스 기능
--------------
쳇 기능
--------------
1. 기본적인 대화(방송)
2. 귓속말 대화(일대일 전송)
3. 대화내용 파일 저장(갈무리),열기
4. 강퇴(옵션)
5. 접속자 목록 출력(옵션)
--------------
파일 기능
--------------
1. 파일을 서버에 등록 
2. 클라이언트 파일 다운
3. 다운 진행상태(진행바) 및 결과 보고(출력)
4. 클라이언트 파일 다운(이어받기, 옵션)
--------------
서비스화면
--------------
1. GUI
2. 메인창 : JSplitPane기반 다중분할창(옵션)
